How did we handle translation?

Hack Your Future (Copenhagen)

Samyar Modabber - June 2019

We try several ways and finally ...


We use react-i18next and i18next.

www.i18next.com www.react.i18next.com

What we need?

install this two node packages:

npm install i18next
npm install react-i18next
How we setup i18next

Add i18n.js (src->react)

and import it in index.js

Add two JSON files:

dk.json and en.json

(src->react->locales)

Add ChangeLanguageBtn components

for toggle the language

(src->react->components)

How we applied to a component

import { withTranslation } from 'react-i18next;

A) Class Component:
define t inside the render function
const { t } = this.props;

B) function component:
gets the t function via ,props .
function AboutShow ({ t }){ ... }

Export component by withTranslation
export default withTranslation()(MyComponent);

Change static text to a dynamic

Before: Sign in

After: {t("sign-in")}

Add key-value to json files

"sign-in": "Sign in" in (en.json)
"sign-in": "Log ind" in (dk.json)
